# Mangalam HDPE Pipes — Product Page

A responsive product landing page built as part of the Gushwork Frontend Assignment.

## 🔗 Live Preview
> Open `index.html` in any modern browser — no build step required.

## 📁 File Structure
```
├── index.html       # Main HTML
├── styles.css       # All styles (responsive, 360px → 1600px)
├── script.js        # Interactive JS (carousel, sticky header, FAQ, modals)
└── assets/          # Product images and icons
```

## ✨ Features
- **Sticky Info Bar** — slides in below the nav after scrolling past the hero section
- **Image Carousel with Zoom** — arrow/thumbnail navigation + cursor-tracked hover zoom
- **FAQ Accordion** — single-open, animated expand/collapse
- **Process Tabs** — tabbed manufacturing stage selector with prev/next navigation
- **Applications Carousel** — horizontally scrollable card strip
- **Modal System** — catalogue request & callback quote modals with form validation
- **Fully Responsive** — optimized across 360px, 767px, 800px, 1080px, 1200px, 1440px, and 1600px breakpoints

## 🛠️ Tech Stack
- **HTML5** — semantic elements throughout
- **CSS3** — custom properties, Flexbox, Grid, no frameworks
- **Vanilla JS** — IntersectionObserver, event delegation, no dependencies

## 🚀 How to Run
Just open `index.html` in a browser. No server or build tools needed.
